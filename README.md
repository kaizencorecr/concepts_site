Concepts Site
===================


Concepts - Equipo colaborativo con iniciativas que impulsan y motivan a emprendedores e innovadores en Costa Rica

*Concepts - Collaborative Team who want to motivate and boost entrepreneurs and innovative people in Costa Rica*

----------


Contexto / Background
-------------

En su primera etapa se tiene únicamente pensado crear un sitio web informativo, que sirva como herramienta para dar una mayor exposición del proyecto y sus diferentes aristas de acción, así crear un medio de comunicación donde se promueva la interacción con el público meta.

*At the first step of the development project, we have agreed to create an informative website in order to create a way to let people know more of Concepts project and their different fields of action. Also, the website is going to have an important role in the communication with their people.*
